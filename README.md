# BulletZone 🔫⚡

**BulletZone** is a fast-paced, wave-based arcade shooter designed for modern web browsers. It combines retro-inspired aesthetics with modern progression systems, challenging players to survive endless waves of enemies while unlocking a diverse arsenal of weapons.

## 🕹️ Core Experience
* **High-Octane Survival:** Combat-focused gameplay where success is measured by kills, waves cleared, and high scores.
* **Weapon Progression:** Start with a basic pistol and earn your way toward high-tier weaponry.
* **Persistent Statistics:** A comprehensive tracking system for total games played, total kills, and career milestones.

## 🛠️ Technical Overview
* **Platform:** Web (HTML5 / CSS3 / JavaScript).
* **Architecture:** Event-driven UI and asynchronous storage management.
* **Rendering:** Clean, performance-optimized DOM and Canvas interaction.

## ⌨️ Controls
* **Mouse/Touch:** Aim and Navigation.
* **LMB/Tap:** Fire and Interaction.
* **System:** Built-in toggles for Music and SFX to customize the sensory experience.
